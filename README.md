## Quick Start

### Requirements

* [elm](http://elm-lang.org/)
* [npm](https://nodejs.org/en/)

### Installation
```
npm install
```

### Running
```
$ node_modules/elm-live/bin/elm-live.js App.elm --output elm.js
```
Go fullscreen with the enter key, or resize the window to fill the screen.
